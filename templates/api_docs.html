📡 API 文档

TextDB Pro 提供简洁的 REST API，方便程序化操作

POST/api/create

创建文本分享

curl -X POST http://YOUR_HOST/api/create \\
  -F "content=Hello World" \\
  -F "ttl=1h" \\
  -F "password=optional_pwd"
// Response
{"success": true, "id": "a1b2c3d4", "url": "http://YOUR_HOST/a1b2c3d4"}
POST/api/upload

上传文件 (最大 50MB)

curl -X POST http://YOUR_HOST/api/upload \\
  -F "file=@/path/to/file" \\
  -F "ttl=1d" \\
  -F "password=optional_pwd"
GET/api/raw/{id}

获取文本原始内容（需带 ?pwd=xxx 如果有密码）

GET/api/file/{id}

下载文件（可选 ?t=download 触发浏览器下载）

GET/api/stats

获取统计信息

{"total": 42, "views": 128}
POST/api/unlock/{id}

验证密码获取访问令牌

{"password": "xxx"} → {"success": true}
